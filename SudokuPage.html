Sudoku SAT Solver

Puzzluoku is an advanced SAT-based solver for Sudoku puzzles, combining algorithmic logic with a user-friendly interface. It is designed to handle complex puzzles efficiently while providing an engaging, minimalist experience for users.

Introduction & Background

In the evolving field of digital solutions for logic-based entertainment, the intersection of user-friendly software and sophisticated computational algorithms presents a significant opportunity for innovation. My dissertation introduces "Puzzluoku," an application designed to transform the experience of solving Sudoku puzzles through a SAT solver. This software is not merely a utility for automating puzzle solving but a comprehensive platform aimed at enhancing user interaction with one of the most popular puzzles globally. The initiative behind Puzzluoku is rooted in a dual objective: to simplify the resolution of puzzles through advanced algorithmic support and to enrich the user experience by offering an intuitive and interactive interface.

Problem Statement and Project Motivation

Sudoku puzzles challenge solvers to fill a grid with digits under strict constraints. Traditional methods, either manual or generic solvers, do not cater to nuanced variations of Sudoku that enthusiasts often encounter. Puzzluoku addresses these gaps by providing a tailored, efficient, and engaging solution that appeals to a broad spectrum of Sudoku enthusiasts, from novices to seasoned experts.

Programming Choice

The code for the "Puzzluoku" Sudoku solver project is written in C#. This choice provides a robust and versatile environment with strong support for object-oriented programming, a rich set of libraries for data structures and logical operations, and excellent debugging tools within Visual Studio. It also integrates well with the .NET framework for creating intuitive user interfaces.

Background: Sudoku and Computational Complexity

Sudoku is a logic-based number placement puzzle that has captured interest for its mathematical structure and computational complexity. Sudoku is NP-complete, meaning there is no known polynomial-time algorithm to solve every puzzle, and solving general Sudoku can be reduced to the 3-SAT problem. This makes the problem theoretically and practically challenging.

Algorithmic Approaches to Solving Sudoku

Given its NP-completeness, various strategies exist: backtracking, brute-force, constraint propagation, and SAT solver adaptations. Puzzluoku transforms Sudoku into a SAT problem using polynomial 3-SAT reduction, where each cell is a Boolean variable and each clause represents constraints, enabling SAT solvers to efficiently tackle even challenging puzzles.

Relevant Research

The project draws from multiple studies including:

  • Ant Colony Optimization (ACO) applied to NP-complete problems
  • Backtracking, simulated annealing, and alternating projection methods
  • Polynomial 3-SAT reduction and SAT encodings for Sudoku
  • Heuristic solutions and advanced SAT solving techniques for difficult puzzles
  • Tools like SUDOKUSAT for classifying puzzle difficulty

Sudoku as a SAT Problem

Sudoku puzzles can be represented as SAT problems by encoding each cell as a Boolean variable and each constraint as clauses in a Boolean formula. Minimal and extended encodings optimize solver efficiency, with complex puzzles benefiting from additional clauses that improve performance. This approach allows advanced SAT techniques—like clause learning and heuristic-based variable selection—to solve difficult puzzles efficiently.
